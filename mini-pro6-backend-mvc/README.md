
-> What This Project Does

   -Runs an Express server
   -Uses MVC (Routes + Controllers)
   -Handles GET and POST requests
   -Sends and receives JSON data

-> Project Structure

project/
├── controllers/
├── routes/
├── app.js
├── package.json

->Routes

GET / → Welcome message

GET /about → About page response

GET /contact → Contact response

GET /time → Current server time

POST /echo → Accepts JSON and returns it
